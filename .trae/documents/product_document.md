# 结婚周年纪念网站 - 产品文档

## 项目概述
为庆祝结婚一周年，创建一个移动端时间轴网站，记录从相识到结婚周年的8个重要时刻。

## 功能需求

### 核心功能
1. **时间轴展示**
   - 8个时间节点：相识、相知、想念、同居、结婚、领证、怀孕、结婚一周年
   - 横向滑动切换或纵向滚动
   - 每页展示精美图片和标题

2. **日记功能**
   - 每个节点可点击查看详细日记
   - 弹窗展示日记内容
   - 温馨的文字排版

3. **移动端优化**
   - 响应式设计
   - 触摸滑动支持
   - 适配各种手机屏幕

### 技术需求
- React + TypeScript
- 移动端优先设计
- 流畅的动画效果
- 图片懒加载优化

## 用户体验
- 温馨浪漫的视觉风格
- 流畅的页面切换动画
- 沉浸式的浏览体验
- 易于分享的链接

## 内容结构
每个时间节点包含：
- 主图（高质量照片）
- 标题（简洁有力）
- 日记（详细记录当时的心情和故事）

## 技术栈
- 前端：React + TypeScript + Tailwind CSS
- 状态管理：Zustand
- 动画：CSS动画 + React过渡
- 部署：Vercel

## 项目结构
```
src/
├── components/          # 组件
│   ├── Timeline/       # 时间轴组件
│   ├── DiaryModal/     # 日记弹窗
│   └── ImageViewer/    # 图片展示
├── pages/              # 页面
│   └── Home/           # 主页
├── hooks/              # 自定义钩子
├── utils/              # 工具函数
└── data/               # 数据（时间节点内容）
```
